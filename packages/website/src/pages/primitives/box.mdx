# Box

All components in Fannypack are built on top of the `<Box>` primitive. Any component in Fannypack can use Box's props. [Read more here](https://reakit.io/docs/box/).

By default it renders a `<div>` element with basic reset styles.

## Import

```js
import { Box } from 'fannypack';
```

## Usage

```jsx-live
<Box>This is a box</Box>
```

## Composition

### `use` prop

```jsx-live
<Box use={Paragraph}>This is a Box rendered as a paragraph</Box>
```

### Hooks

```jsx
const boxProps = Box.useProps();

<p {...boxProps}>This is a Box rendered as a paragraph.</p>
```

### Render props

```jsx-live
<Box>{boxProps => <p {...boxProps}>This is a Box rendered as a paragraph</p>}</Box>
```

## CSS props

You're able to apply any CSS attribute as a prop to `<Box>`.

```jsx-live
<Box backgroundColor="primaryTint" color="primary800" padding="major-2">This is a box</Box>
```
