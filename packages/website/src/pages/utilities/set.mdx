# Set

## Import

```jsx
import { Set } from 'fannypack';
```

## Usage

A `<Set>` can be used to list a set of components. It will automatically space out the components, and wrap them when they reach the width of the viewport.

```jsx-live
<Set>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
  <Button>Button 4</Button>
  <Button>Button 5</Button>
  <Button>Button 6</Button>
  <Button>Button 7</Button>
</Set>
```

### Spacing

You can modify the spacing of the set with the `spacing` prop.

```jsx-live
<Set spacing="minor-1">
  <Tag>Tag 1</Tag>
  <Tag>Tag 2</Tag>
  <Tag>Tag 3</Tag>
  <Tag>Tag 4</Tag>
  <Tag>Tag 5</Tag>
  <Tag>Tag 6</Tag>
  <Tag>Tag 7</Tag>
</Set>
```

### Vertical

You can also have a vertical set with the `isVertical` prop.

```jsx-live
<Set isVertical spacing="minor-1">
  <Tag>Tag 1</Tag>
  <Tag>Tag 2</Tag>
  <Tag>Tag 3</Tag>
  <Tag>Tag 4</Tag>
  <Tag>Tag 5</Tag>
  <Tag>Tag 6</Tag>
  <Tag>Tag 7</Tag>
</Set>
```

By default, `<Set>` will preserve the width of it's children. However, if you'd like your children to fill the width of `<Set>`, use the `isFilled` prop:

```jsx-live
<Set isVertical isFilled>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
  <Button>Button 4</Button>
  <Button>Button 5</Button>
  <Button>Button 6</Button>
  <Button>Button 7</Button>
</Set>
```

### Responsive breakpoint

A set can snap from horizontal to vertical after a given breakpoint with the `verticalBreakpoint` prop.

```jsx-live
<Set verticalBreakpoint="max-tablet">
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
  <Button>Button 4</Button>
  <Button>Button 5</Button>
  <Button>Button 6</Button>
  <Button>Button 7</Button>
</Set>
```
