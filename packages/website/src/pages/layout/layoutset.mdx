# LayoutSet

## Import

```jsx
import { LayoutSet } from 'fannypack';
```

### Basic Usage

The `<LayoutSet>` component is a utility belt that wraps around layout elements to space them out correctly. It uses the [`<Set>` utility](/utilities/set) internally.
