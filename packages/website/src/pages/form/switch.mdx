# Switch

## Import

```jsx
import { Switch, SwitchField } from 'fannypack'
```

## Usage

```jsx-live
<Switch label="Check me" />
```

### Disabled

Make the switch disabled with the `disabled` prop.

```jsx-live
<Switch disabled label="Click me" />
<Switch checked disabled label="Click me" />
```

### Colors

Add colors to your switch by using the `palette` prop.

```jsx-live
<Switch defaultChecked palette="secondary" label="Check me" />
<Switch defaultChecked palette="success" label="Check me" />
<Switch defaultChecked palette="danger" label="Check me" />
<Switch defaultChecked palette="warning" label="Check me" />
```

### States

A switch can use different states (as per palette) such as `danger`, `success` and `warning`.

```jsx-live
<Switch state="danger" label="Click me" />
<Switch state="success" label="Click me" />
<Switch state="warning" label="Click me" />
<Switch state="primary" label="Click me" />
```

### Fields

The `<SwitchField>` component contains of the [`<Switch>` component](#input) wrapped by the [`<FieldWrapper>` component](/form/fieldwrapper).

It accepts a combination of [`<Switch>` props](#TODO) and [`<FieldWrapper>` props](#TODO).

```jsx-live
<SwitchField
  switchLabel="Check me"
  label="Check below to subscribe"
/>
```

### Controlled Usage

The previous examples are examples of uncontrolled usage. To control the value yourself, use a combination of `onChange` and `value`.

Note: Do not use `defaultValue` with controlled inputs!

```function-live
function Example() {
  const [checked, setChecked] = React.useState(true);

  return (
    <Switch
      label="Click me"
      checked={checked}
      onChange={e => setChecked(e.target.checked)}
    />
  );
}
```
