# Text

### Import

```jsx
import { Text } from 'fannypack'
```

### Basic Usage

The `<Text>` component renders a `<span>` with basic reset styles by default.

```.jsx
<Text>Hello world!</Text>
```

> Remember, you can also specify different colors with the `color` prop.

```.jsx
<Text color="success">Hello world!</Text><br />
<Text color="danger">Hello world!</Text>
```
