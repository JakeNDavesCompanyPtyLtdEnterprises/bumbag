# Heading

### Import

```jsx
import { Heading } from 'fannypack'
```

### Basic Usage

By default, `<Heading>` renders a `<h1>` with default margins applied.

```.jsx
<Heading>Heading 1</Heading>
<p>Hello world!</p>
```

> Remember, you can also specify different colors with the `color` prop.

```.jsx
<Heading color="success">Heading 1</Heading>
<Heading color="danger">Heading 1</Heading>
```

### Other headers

Use the `use` prop to change from the default `h1` element to `h2`, `h3`, `h4`, `h5`, `h6`, `span`, or others.

```.jsx
<Heading>Heading 1</Heading>
<Heading use="h2">Heading 2</Heading>
<Heading use="h3">Heading 3</Heading>
<Heading use="h4">Heading 4</Heading>
<Heading use="h5">Heading 5</Heading>
<Heading use="h6">Heading 6</Heading>
```

### Subheadings

Subheading's are the same size as a normal heading, however have a slightly lower font weight.

```.jsx
<Heading isSubHeading>Heading 1</Heading>
<Heading use="h2" isSubHeading>Heading 2</Heading>
<Heading use="h3" isSubHeading>Heading 3</Heading>
<Heading use="h4" isSubHeading>Heading 4</Heading>
<Heading use="h5" isSubHeading>Heading 5</Heading>
<Heading use="h6" isSubHeading>Heading 6</Heading>
```

A header can also be easily grouped with a subheading

```.jsx
<Heading>Jake's Fannypack</Heading>
<Heading use="h3" isSubHeading>Simply the best</Heading>
<Heading use="h3">Jake's Fannypack</Heading>
<Heading use="h5" isSubHeading>Simply the best</Heading>
```
