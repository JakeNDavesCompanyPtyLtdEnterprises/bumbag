# Content

The `Page.Content` component is a template component that helps you with creating a 'plain' page with content. It comes with margins & paddings built in.

### Import

```js
import { Page } from 'fannypack'
```

### Breakpoints

You can set a default breakpoint on all `Page.Content` components via the Fannypack theme:

```jsx
import { ThemeProvider } from 'fannypack';

const theme = {
  Page: {
    Content: {
      defaultProps: {
        breakpoint: 'desktop'
      }
    }
  }
}

const App = () => (
  <ThemeProvider theme={theme}>
    // ... your app
  </ThemeProvider>
);
```

Or, you can set it with the `breakpoint` prop on each component:
