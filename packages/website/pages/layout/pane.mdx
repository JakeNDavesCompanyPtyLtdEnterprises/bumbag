# Pane

### Import

`import { Pane } from 'fannypack'`

### Basic Usage

A `<Pane>` is the same as a `<Box>`, except it has a default border radius, and has the ability to add different types of borders to it. Using `<Pane>` and `<Box>` together are most likely the two building blocks for the layout of your app.

```.jsx
<Pane backgroundColor="white700" padding="major-4">
  Hello world
</Pane>
```

### Shadowed

```.jsx
<Pane border="shadow" padding="major-4">Hello world</Pane>
```

### Bordered

```.jsx
<Pane border padding="major-4">Hello world</Pane>
```

### Elevations

```.jsx
<Set spacing="major-4">
  <Pane elevation="100" padding="major-4">Hello world</Pane>
  <Pane elevation="200" padding="major-4">Hello world</Pane>
  <Pane elevation="300" padding="major-4">Hello world</Pane>
  <Pane elevation="400" padding="major-4">Hello world</Pane>
</Set>
```

### Full width

```.jsx
<Pane border isFullWidth padding="major-4">Hello world</Pane>
```

### Theming

#### Schema

```jsx
{
  Pane: {
    base: css | Object,

    // Styles to apply for bordered panes
    border: {
      default: css | Object,
      shadow: css | Object
    },

    // Scales for pane elevations, maps to a number (pixels). These keys below are required, but you can also add more.
    elevations: {
      100: number, // default = 4
      200: number, // default = 8
      300: number, // default = 12
      400: number, // default = 24
    }
  }
}
```
