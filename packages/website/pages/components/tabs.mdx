# Tabs

### Import

```jsx
import { Tabs } from 'fannypack'
```

### Basic Usage

By default, `<Tabs>` renders a `<ul>` that consists of `<Tabs.Tab>` which render `<li>` elements.

Fannypack also has a `<Tabs.Container>` component which you can use to control your tab state.

```.jsx
<Tabs.Container>
  {tabs => (
    <Tabs>
      <Tabs.Tab tab="dogs" {...tabs}>
        Dogs
      </Tabs.Tab>
      <Tabs.Tab tab="cats" {...tabs}>
        Cats
      </Tabs.Tab>
      <Tabs.Tab tab="parrots" {...tabs}>
        Parrots
      </Tabs.Tab>
    </Tabs>
  )}
</Tabs.Container>
```

### Tab content

```.jsx
<Tabs.Container>
  {tabs => (
    <React.Fragment>
      <Tabs>
        <Tabs.Tab tab="dogs" {...tabs}>
          Dogs
        </Tabs.Tab>
        <Tabs.Tab tab="cats" {...tabs}>
          Cats
        </Tabs.Tab>
      </Tabs>
      <Tabs.Panel tab="dogs" padding="major-3" {...tabs}>
        Dogs are awesome
      </Tabs.Panel>
      <Tabs.Panel tab="cats" padding="major-3" {...tabs}>
        Cats are meh
      </Tabs.Panel>
    </React.Fragment>
  )}
</Tabs.Container>
```

### Fitted tabs

To make the tabs span the width of the element. Use the `isFitted` prop.

```.jsx
<Tabs.Container>
  {tabs => (
    <Tabs isFitted>
      <Tabs.Tab tab="dogs" {...tabs}>
        Dogs
      </Tabs.Tab>
      <Tabs.Tab tab="cats" {...tabs}>
        Cats
      </Tabs.Tab>
      <Tabs.Tab tab="parrots" {...tabs}>
        Parrots
      </Tabs.Tab>
    </Tabs>
  )}
</Tabs.Container>
```

### Alignment

Tabs can also be center or right aligned.

#### Center aligned

```.jsx
<Tabs.Container>
  {tabs => (
    <Tabs align="center">
      <Tabs.Tab tab="dogs" {...tabs}>
        Dogs
      </Tabs.Tab>
      <Tabs.Tab tab="cats" {...tabs}>
        Cats
      </Tabs.Tab>
      <Tabs.Tab tab="parrots" {...tabs}>
        Parrots
      </Tabs.Tab>
    </Tabs>
  )}
</Tabs.Container>
```

#### Right aligned

```.jsx
<Tabs.Container>
  {tabs => (
    <Tabs align="right">
      <Tabs.Tab tab="dogs" {...tabs}>
        Dogs
      </Tabs.Tab>
      <Tabs.Tab tab="cats" {...tabs}>

      </Tabs.Tab>
      <Tabs.Tab tab="parrots" {...tabs}>
        Parrots

```

### Icons

**Note:** To use tab icons, you will need to follow the [instructions to set up icons with Fannypack](/components/icon).

```.jsx
<Tabs.Container>
  {tabs => (
    <Tabs>
      <Tabs.Tab tab="documents" {...tabs}>
        <Icon icon="solid-file" marginRight="major-1" /> Documents
      </Tabs.Tab>
      <Tabs.Tab tab="messages" {...tabs}>
        <Icon icon="solid-comments" marginRight="major-1" /> Messages
      </Tabs.Tab>
      <Tabs.Tab tab="profile" {...tabs}>
        <Icon icon="solid-user" marginRight="major-1" /> Profile
      </Tabs.Tab>
    </Tabs>
  )}
</Tabs.Container>
```

### Accessibility

You can use the keyboard arrow buttons to navigate between the tabs.

```.jsx
<Tabs.Container>
  {tabs => (
    <Tabs>
      <Tabs.Tab tab="dogs" {...tabs}>
        Dogs
      </Tabs.Tab>
      <Tabs.Tab tab="cats" {...tabs}>
        Cats
      </Tabs.Tab>
      <Tabs.Tab tab="parrots" {...tabs}>
        Parrots
      </Tabs.Tab>
    </Tabs>
  )}
</Tabs.Container>
```
