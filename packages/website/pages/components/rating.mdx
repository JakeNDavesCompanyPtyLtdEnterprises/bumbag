# Rating

### Import

```jsx
import { Rating } from 'fannypack'
```

### Basic Usage

The rating callback is called with `{ rating, maxRating }`.

```.jsx
<Rating onRate={result => console.log(result)}/>
```

### Default Rating

```.jsx
<Rating defaultRating={3} />
```

### Disabled

```.jsx
<Rating defaultRating={3} disabled />
```

### Sizes

```.jsx
<Rating defaultRating={1} size="small" />
<Rating defaultRating={2} />
<Rating defaultRating={3} size="medium" />
<Rating defaultRating={4} size="large" />
```

### Max Rating

```.jsx
<Rating maxRating={1} />
<Rating maxRating={2} />
<Rating maxRating={3} />
<Rating maxRating={4} />
<Rating />
<Rating maxRating={10} />
```
