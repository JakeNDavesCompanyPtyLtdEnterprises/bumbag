# Spinners

### Import

```jsx
import { Spinner } from 'fannypack'
```

### Basic Usage

```.jsx
<Spinner />
```

### Sizes

```.jsx
<Spinner size="small" />
<Spinner marginLeft="major-1" />
<Spinner size="medium" marginLeft="major-1" />
<Spinner size="large" marginLeft="major-1" />
```

### Colors

```.jsx
<Spinner color="success" />
<Spinner color="danger" marginLeft="major-1" />
<Spinner color="warning" marginLeft="major-1" />
```
