# Avatar

### Import

```jsx
import { Avatar } from 'fannypack'
```

### Basic Usage

```.jsx
<Avatar
  src="https://raw.githubusercontent.com/bigbellies/fannypack/master/src/_docs/bean.jpg"
  alt="Bean"
  fit="cover"
  height={100}
  width={100}
/>
```

### Avatar shapes

Avatars are available in two different shapes and can be changed using the `kind` prop.

```.jsx
<Avatar
  src="https://raw.githubusercontent.com/bigbellies/fannypack/master/src/_docs/bean.jpg"
  kind="circle"
  alt="Bean"
  fit="cover"
  height={100}
  width={100}
/>
```

### Avatar using initials

Avatars can also be initials using the `initials` prop.

```.jsx
<Set>
  <Avatar
    palette="primary"
    initials="MB"
    alt="Bean"
    kind="circle"
    size="xsmall"
  />
  <Avatar
    palette="success"
    initials="Mr Bean"
    alt="Bean"
    kind="circle"
    size="small"
  />
  <Avatar
    palette="primary"
    initials="MB"
    alt="Bean"
  />
  <Avatar
    palette="danger"
    initials="MB"
    alt="Bean"
    kind="circle"
    size="medium"
  />
  <Avatar
    palette="warning"
    initials="MB"
    alt="Bean"
    kind="circle"
    size="large"
  />
</Set>
```
