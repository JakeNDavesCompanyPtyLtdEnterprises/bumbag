# Navigation

### Import

```jsx
import { Navigation } from 'fannypack';
```

### Basic Usage

By default, `<Navigation>` renders a `<nav>`.

```.jsx
<Navigation>
  <List>
    <List.Item>
      <Link href="https://google.com">Google</Link>
    </List.Item>
    <List.Item>
      <Link href="https://github.com">GitHub</Link>
    </List.Item>
  </List>
</Navigation>
```

You can also have a horizontal list by doing:

```.jsx
<Navigation>
  <List isHorizontal>
    <List.Item>
      <Link href="https://google.com">Google</Link>
    </List.Item>
    <List.Item>
      <Link href="https://github.com">GitHub</Link>
    </List.Item>
  </List>
</Navigation>
```

### Accessibility

Using an accessibility title (`a11yTitle`), will enable screen readers to read the contents of the title.

```.jsx
<Navigation a11yTitle="Awesome sites">
  <List>
    <List.Item>
      <Link href="https://google.com">Google</Link>
    </List.Item>
    <List.Item>
      <Link href="https://github.com">GitHub</Link>
    </List.Item>
  </List>
</Navigation>
```
