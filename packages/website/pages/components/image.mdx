# Image

### Import

```jsx
import { Image } from 'fannypack'
```

### Basic Usage

`<Image>` renders a `<img>` with `max-width: 100%`  and default reset styles applied.

```.jsx
<Image
  src="/static/bean.jpg"
  alt="Bean"
  width={500}
/>
```

`<Image>` is also responsive by default, however, if you want to fix the width, add the `isFixed` prop.

```.jsx
<Image
  isFixed
  src="/static/bean.jpg"
  alt="Bean"
  width={500}
/>
```

### Image fit

Setting a `fit` prop to `"contain"` means that the whole image will fit in the bounds and it will not lose its scale.

```.jsx
<Image
  fit="contain"
  height={300}
  width={300}
  src="/static/bean.jpg"
  alt="Bean"
  backgroundColor="whitesmoke"
/>
```

Setting a `fit` prop to `"cover"` means that the image will fill the bounds and it will not lose its scale.

```.jsx
<Image
  fit="cover"
  height={300}
  width={300}
  src="/static/bean.jpg"
  alt="Bean"
  backgroundColor="whitesmoke"
/>
```

#### Positioning

You can change the position of a fitted image with the `fitPosition` prop. This prop mimics the `object-position` CSS attribute.

```.jsx
<Image
  fit="contain"
  fitPosition="top"
  height={300}
  width={300}
  src="/static/bean.jpg"
  alt="Bean"
  backgroundColor="whitesmoke"
/>
```

```.jsx
<Image
  fit="cover"
  fitPosition="10px 5px"
  height={300}
  width={300}
  src="/static/bean.jpg"
  alt="Bean"
  backgroundColor="whitesmoke"
/>
```

### Theming

#### Schema

```jsx
{
  Image: {
    base: css | Object,

    // Style to apply to fixed images
    fixed: css | Object,

    // Styles to apply when `fit` prop is present
    fit: {
      base: css | Object,
      cover: css | Object,
      contain: css | Object
    }
  }
}
```
