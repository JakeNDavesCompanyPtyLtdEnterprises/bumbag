# Callout

### Import

```jsx
import { Callout } from 'fannypack'
```

### Basic Usage

The `<Callout>` component uses `<Card>` internally, and is used to highlight important content to users.

If the content you are displaying is **brief**, **important** & requires the user's **immediate attention**, consider using the [`<Alert>` component](/components/alert)

You can render different intents for the callout using the `type` prop. A Callout can take a type of `"info"`, `"success"`, `"warning"` or `"danger"`.

```.jsx
<Callout type="info" title="This is a title" marginBottom="20px">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Callout>
<Callout type="success" title="This is a title" marginBottom="20px">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Callout>
<Callout type="danger" title="This is a title" marginBottom="20px">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Callout>
```

### Footer

You can render a footer for the callout using the `footer` prop.

```.jsx
<Callout type="info" title="This is a title" footer={<ActionButtons justifyContent="flex-end" width="100%" submitProps={{ palette: 'info' }} />}>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Callout>
```

### Close buttons

To add a close button, add the `showCloseButton` & `onClickClose` props.

**Note:** You can also add props to the close button with `closeButtonProps`. It takes [`<Button>`'s props](/components/button)

```.jsx
<Callout type="info" title="This is a title" onClickClose={() => alert('Closed!')} showCloseButton marginBottom="20px">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue, ultrices
  eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum consectetur lectus
  augue sit amet justo.
</Callout>
```

### Theming

#### Schema

```jsx
{
  Callout: {
    base: css | Object,

    // Style to apply for tinted callouts
    tint: css | Object,

    // Style to apply for callout content
    Content: {
      base: css | Object
    },

    // Style to apply for callout header
    Header: {
      base: css | Object
    },

    // Style to apply for callout footer
    Footer: {
      base: css | Object
    },

    // Style to apply for callout title
    Title: {
      base: css | Object
    },

    // Style to apply for callout icon
    Icon: {
      base: css | Object
    },

    // Style to apply for callout close button
    Close: {
      base: css | Object,
      hover: css | Object
    }
  }
}
```
