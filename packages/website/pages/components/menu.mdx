# Menu

### Import

```jsx
import { Menu } from 'fannypack'
```

### Basic Usage

The `<Menu>` component is used with `<Menu.Group>` and `<Menu.Item>` to create a menu. These components create
an accessible menu with the ability to:

- Navigate through the menu with the **Up** & **Down** arrow keys (or **Left** & **Right**),
- Go to the first item of the menu with the **Home** (or **fn + Up**) key,
- Go to the last item of the menu with the **End** (or **fn + Down**) key.

```.jsx
<Menu a11yTitle="Main menu">
  <Menu.Group>
    <Menu.Item
      icon="solid-pen"
      onClick={() => alert('Edit')}
    >
      Edit...
    </Menu.Item>
    <Menu.Item icon="solid-share" onClick={() => alert('Share')}>
      Share...
    </Menu.Item>
    <Menu.Item isDisabled icon="solid-file-signature" onClick={() => alert('Rename')}>
      Rename...
    </Menu.Item>
    <Menu.Item color="danger" icon="solid-trash-alt" onClick={() => alert('Delete')}>
      Delete...
    </Menu.Item>
  </Menu.Group>
  <Menu.Divider />
  <Menu.Group>
    <Menu.Item use={Link} icon="solid-external-link-alt" href="https://google.com">
      Go to Google
    </Menu.Item>
  </Menu.Group>
</Menu>
```

### Active items

```.jsx
<Menu a11yTitle="Main menu">
  <Menu.Group>
    <Menu.Item>
      Home
    </Menu.Item>
    <Menu.Item isActive>
      Account
    </Menu.Item>
    <Menu.Item>
      Logout
    </Menu.Item>
  </Menu.Group>
</Menu>
```

### Grouped menus

A menu can be grouped (and also titled) using the `<Menu.Group>` component:

```.jsx
<Menu a11yTitle="Main menu">
  <Menu.Group title="Actions">
    <Menu.Item icon="solid-pen" onClick={() => alert('Edit')}>
      Edit...
    </Menu.Item>
    <Menu.Item icon="solid-share" onClick={() => alert('Share')}>
      Share...
    </Menu.Item>
    <Menu.Item icon="solid-file-signature" onClick={() => alert('Rename')}>
      Rename...
    </Menu.Item>
    <Menu.Item color="danger" icon="solid-trash-alt" onClick={() => alert('Rename')}>
      Delete...
    </Menu.Item>
  </Menu.Group>
  <Menu.Divider />
  <Menu.Group title="Links">
    <Menu.Item use={Link} icon="solid-external-link-alt" href="https://google.com">
      Go to Google
    </Menu.Item>
  </Menu.Group>
</Menu>
```

### Horizontal menu

You can change the orientation of the menu to horizontal by adding the `isHorizontal` prop:

```.jsx
<Menu a11yTitle="Main menu" isHorizontal>
  <Menu.Group>
    <Menu.Item icon="solid-pen" onClick={() => alert('Edit')}>
      Edit...
    </Menu.Item>
    <Menu.Item icon="solid-share" onClick={() => alert('Share')}>
      Share...
    </Menu.Item>
    <Menu.Item icon="solid-file-signature" onClick={() => alert('Rename')}>
      Rename...
    </Menu.Item>
    <Menu.Item color="danger" icon="solid-trash-alt" onClick={() => alert('Delete')}>
      Delete...
    </Menu.Item>
  </Menu.Group>
</Menu>
```

### Dropdown menu

You can even have a dropdown menu by using `<Popover>`.

If the menu button is focused, you can show the menu by pressing the **Down**, **Space** or **Enter** keys.

```.jsx
<Menu.Popover
  content={
    <Menu a11yTitle="Main menu">
      <Menu.Group title="Actions">
        <Menu.Item icon="solid-pen" onClick={() => alert('Edit')}>
          Edit...
        </Menu.Item>
        <Menu.Item icon="solid-share" onClick={() => alert('Share')}>
          Share...
        </Menu.Item>
        <Menu.Item icon="solid-file-signature" onClick={() => alert('Rename')}>
          Rename...
        </Menu.Item>
        <Menu.Item color="danger" icon="solid-trash-alt" onClick={() => alert('Delete')}>
          Delete...
        </Menu.Item>
      </Menu.Group>
      <Menu.Divider />
      <Menu.Group title="Links">
        <Menu.Item use={Link} icon="solid-external-link-alt" href="https://google.com">
          Go to Google
        </Menu.Item>
      </Menu.Group>
    </Menu>
  }
>
  <Menu.Button>Toggle menu</Menu.Button>
</Menu.Popover>
```
