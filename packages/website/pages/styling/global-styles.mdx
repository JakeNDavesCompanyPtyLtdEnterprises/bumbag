# Global Styles

To add global styles to Fannypack, you will need to inject them in the Fannypack theme:

```jsx
import { ThemeProvider, css } from 'fannypack';

const theme = {
  global: {
    fontFamily: 'Comic Sans MS',
    fontSize: 20,
    base: css`
      html,
      body {
        background-color: pink;
      }
    `
  }
}

export default () => (
  <ThemeProvider theme={theme}>
    {/* your app... */}
  </ThemeProvider>
)
```

### Theming

#### Schema

```jsx
{
  global: {
    base: css | Object
  }
}
```
