# Spacing

Fannypack uses two types of scales for its spacing system: the **8px major scale** and the **4px minor scale**.

## The 8px major scale (recommended)

The **8px major scale** is most commonly used for positional spacing. Like **heights & widths** of components, as well as **margins & paddings** between components on a layout.

**Note:** It is recommended to try stick to the 8px scale unless you need more precision. In that case, use the **4px minor scale**.

### Usage via CSS props

```.jsx
<FieldSet isHorizontal spacing="major-2">
  <InputField label="First name" />
  <InputField label="Last name" />
</FieldSet>
<Button marginTop="major-2">Save</Button>
```

### Usage via styled components

```jsx
import { space } from 'fannypack';

const SaveButton = styled(Button)`
  margin-top: ${space(2, 'major')}rem;
`;
```

## The 4px minor scale

The **4px minor scale** is most commonly used for precise spacing (**margins & paddings**) of atomic components (e.g. the inner padding of a checkbox).

Fannypack also exports a utility function called `space` to help you with calculating the scale values.

## Theming

### Schema

```jsx
{
  layout: {
    minorUnit: number // default = 4,
    majorUnit: number // default = 8
  }
}
```
