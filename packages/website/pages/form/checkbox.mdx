# Checkbox

### Import

```jsx
import { Checkbox } from 'fannypack'
```

### Basic Usage

```.jsx
<Checkbox label="Click me" />
```

### Disabled

Make the checkbox disabled with the `disabled` prop.

```.jsx
<Checkbox disabled label="Click me" />
<Checkbox checked disabled label="Click me" />
```

### Indeterminate

```.jsx
<Checkbox label="Click me" checked indeterminate />
```

### States

A checkbox can use different states (as per palette) such as `danger`, `success` and `warning`.

```.jsx
<Checkbox state="danger" label="Click me" />
<Checkbox state="success" label="Click me" />
<Checkbox state="warning" label="Click me" />
<Checkbox state="primary" label="Click me" />
```

### Controlled Usage

The previous examples are examples of uncontrolled usage. To control the value yourself, use a combination of `onChange` and `checked`.

Note: Do not use `defaultChecked` with controlled inputs!

```.jsx
<Component initialState={{ value: true }}>
  {({ state, setState }) => (
    <Checkbox
      label="Click me"
      checked={state.value}
      onChange={e => setState({ value: e.target.checked })}
    />
  )}
</Component>
```

# CheckboxField

### Import

```jsx
import { CheckboxField } from 'fannypack'
```

### Basic Usage

The `<CheckboxField>` component contains of the [`<Checkbox>` component](#checkbox) wrapped by the [`<FieldWrapper>` component](/form/fieldwrapper).

It accepts a combination of [`<Checkbox>` props](/form/checkbox#props) and [`<FieldWrapper>` props](/form/fieldwrapper#props).

```.jsx
<CheckboxField
  a11yId="isSubscribed"
  checkboxLabel="Check me"
  label="Check below to subscribe"
/>
```
