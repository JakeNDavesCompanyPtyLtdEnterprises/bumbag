# List

### Import

```jsx
import { List } from 'fannypack'
```

### Basic Usage

`<List>` renders a `<ul>` element by default with reset styles and `list-style` unset.

```.jsx
<Paragraph>My awesome list:</Paragraph>
<List>
  <List.Item>Item 1</List.Item>
  <List.Item>Item 2</List.Item>
  <List.Item>Item 3</List.Item>
  <List.Item>Item 4</List.Item>
</List>
```

### Types

Specify if you want your list items to have a style type with the `listStyleType` CSS property.

[Click here](https://developer.mozilla.org/en-US/docs/Web/CSS/list-style-type) to see all the `listStyleType` values.

```.jsx
<Paragraph>My awesome list:</Paragraph>
<List listStyleType="disc" listStylePosition="inside">
  <List.Item>Item 1</List.Item>
  <List.Item>Item 2</List.Item>
</List>
<List listStyleType="circle" listStylePosition="inside">
  <List.Item>Item 1</List.Item>
  <List.Item listStyleType="georgian">Item 2</List.Item>
  <List listStyleType="circle">
    <List.Item>Item 1</List.Item>
    <List.Item>Item 2</List.Item>
  </List>
</List>
```

### Ordered lists

To use an ordered list, use the `isOrdered` prop.

```.jsx
<Paragraph>My awesome list:</Paragraph>
<List isOrdered listStylePosition="inside">
  <List.Item>Item 1</List.Item>
  <List.Item>Item 2</List.Item>
  <List isOrdered>
    <List.Item>Item 3</List.Item>
    <List isOrdered>
      <List.Item>Item 4</List.Item>
    </List>
  </List>
</List>
```

### Horizontal lists

To use an horizontal list, use the `isHorizontal` prop.

```.jsx
<List isHorizontal>
  <List.Item>Item 1</List.Item>
  <List.Item>Item 2</List.Item>
</List>
```

```jsx
  }
}
```
