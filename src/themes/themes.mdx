# Styling & theming components

## Styling

There are **three** methods to style your Fannypack components (similar to [Reakit](https://reakit.io/guide/styling)):

- [Using styled-components](#using-styled-components)
- [Using inline styles](#inline-styles)
- [Using themes](#theming)

<hr style={{ marginTop: '2rem' }}/>

### Using styled-components

Fannypack uses [styled-components](https://www.styled-components.com/docs/basics#getting-started) internally to style components with a `styled` method.

Here is a basic example using `styled` with **Fannypack**:

```jsx
import { Button, styled } from 'fannypack';

const RedButton = styled(Button)`
  background-color: red;
  color: white;
`;

<RedButton>I am a red button!</RedButton>
```

### Inline styles

Alternatively, you can also use inline styles in Fannypack. These inline styles can be passed through as camel-cased props. Inline styling can be very useful for margins and paddings.

Here is an example of using inline styling with **Fannypack**:

<Playground>
  <Box backgroundColor="primary" width="50px" height="50px" />
  <Box backgroundColor="primary" marginTop="medium" width="50px" height="50px" />
</Playground>

#### Colors

When using inline styling, you can specify a color from the palette with any color CSS property.

Example:

<Playground>
  <Box backgroundColor="success" width="50px" height="50px" />
</Playground>

#### Margins & paddings

You can also specify a humanized value for margins & paddings.

<Playground>
  <Box padding="large" marginLeft="xxlarge" marginRight="xxlarge" backgroundColor="primary" color="white">Hello world</Box>
</Playground>

Possible values are: `xxsmall`, `xsmall`, `small`, `medium`, `large`, `xlarge`, `xxlarge`. You can also pass the normal values like `5px`, `1rem`, `auto` as well if you prefer.

#### Font sizes

You can also specify a humanized value for font sizes.

<Playground>
  <Text fontSize="large">Hello world</Text>
</Playground>

Possible values are: `small`, `medium`, `large`, `xlarge`, `xxlarge`. You can also pass the normal values like `16px` and `1.5rem` as well if you prefer.

## Theming

Fannypack can be extensively themed to meet your requirements. You can either **override the default Fannypack theme** or **create your own custom theme** from scratch. If you like the default theme and want to keep it simple, we recommend you just override the default.

### Overriding the default theme

The `<ThemeProvider>` component accepts `theme` as a prop, where you can specify overrides to the default Fannypack theme. The schema of the `theme` object can be seen [here](#theme-config-schema).

[Click here](https://github.com/fannypackui/fannypack/blob/master/src/themes/default.js) to see the default theme.

#### Example

```jsx
import { ThemeProvider, css } from 'fannypack';
import { lighten } from 'polished';

const theme = {
  global: {
    base: css`
      font-family: 'Comic Sans MS'
    `
  },
  palette: {
    primary: lighten(0.2, 'blue')
  },
  layout: {
    mobileBreakpoint: 520,
    tabletBreakpoint: 960
  },
  Button: {
    disabled: css`
      opacity: 0.2
    `
  },
  Text: css`
    font-weight: 300;
  `
}

const App = () => (
  <ThemeProvider theme={theme}>
    // ... your app
  </ThemeProvider>
);
```

### Creating your own theme

If you decide to not use the default Fannypack theme and roll out your own, just add the `isStandalone` flag to `<ThemeProvider>`.

#### Example

```jsx
import { ThemeProvider, defaultTheme } from 'fannypack';

const App = () => (
  <ThemeProvider isStandalone theme={defaultTheme()}>
    // ... your app
  </ThemeProvider>
);
```

#### LayoutThemeConfig

```jsx
{
  mobileBreakpoint: number,
  tabletBreakpoint: number,
  desktopBreakpoint: number,
  widescreenBreakpoint: number,
  fullHDBreakpoint: number,
  gapFactor: number,
  spacing: {
    xxsmall: number,
    xsmall: number,
    small: number,
    medium: number,
    large: number,
    xlarge: number,
    xxlarge: number
  }
}
```

##### Default

```jsx
{
  mobileBreakpoint: 480,
  tabletBreakpoint: 768,
  desktopBreakpoint: 1024,
  widescreenBreakpoint: 1200,
  fullHDBreakpoint: 1440,
  gapFactor: 0.5,
  spacing: {
    xxxsmall: 0.25,
    xxsmall: 0.5,
    xsmall: 1,
    small: 1.5,
    medium: 2,
    large: 2.5,
    xlarge: 3,
    xxlarge: 3.5
    xxxlarge: 4
  }
}
```

#### FontSizeThemeConfig

```jsx
{
  small: number,
  medium: number,
  large: number,
  xlarge: number,
  xxlarge: number,
  xxxlarge: number
}
```

##### Default

```jsx
{

  medium: 1.5,
  large: 2,
  xlarge: 2.5,
  xxlarge: 3
}
```
