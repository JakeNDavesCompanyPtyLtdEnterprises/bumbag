# Callout

## Import

`import { Callout } from 'fannypack'`

## Basic Usage

The `<Callout>` component uses `<Card>` internally, and is used to highlight important content to users. It is recommended to try use the [`<Alert>` component](/components/alert) for displaying content that is of shorter length.

You can render different intents for the callout using the `type` prop. A Callout can take a type of `"info"`, `"success"`, `"warning"` or `"danger"`.
