# Button

## Import

`import { Button } from 'fannypack'`

## Basic Usage

<Playground>
  <Button>Default button</Button>
</Playground>

## Custom elements

Button, by default, renders a `<button>` element.
However, you are able to change it to a custom element using the `as` prop. See more [here](https://reakit.io/components/button).

<Playground>
  <Set>
    <Button>Default</Button>
    <Button use="a" href="#">
      Anchor
    </Button>
  </Set>
</Playground>

## Colors

Colors can be added to a button with the `palette` prop.

<Playground>
  <Set>
    <Button>Default</Button>
    <Button palette="primary">Primary</Button>
    <Button palette="secondary">Secondary</Button>
    <Button palette="success">Success</Button>
    <Button palette="danger">Danger</Button>
    <Button palette="warning">Warning</Button>
  </Set>
</Playground>

## Loading

To indicate a button is loading, use the `isLoading` prop.

<Playground>
  <Set>
    <Button isLoading>Default</Button>
    <Button palette="primary" isLoading>
      Primary
    </Button>
    <Button palette="primary" isLoading kind="link">
      Primary
    </Button>
    <Button palette="success" isLoading kind="outlined">
      Success
    </Button>
  </Set>
</Playground>

## Outlines

An outline can be placed on a button by assigning `"outlined"` to the `kind` prop.

<Playground>
  <Set>
    <Button palette="primary" kind="outlined">
      Primary
    </Button>
    <Button palette="success" kind="outlined">
      Success
    </Button>
    <Button palette="danger" kind="outlined">
      Danger
    </Button>
  </Set>
</Playground>

## Sizes

Buttons are available in four different sizes and can be added with a `size` prop.

<Playground>
  <Set>
    <Button size="small">Small</Button>
    <Button>Default</Button>
    <Button size="medium">Medium</Button>
    <Button size="large">Large</Button>
  </Set>
</Playground>

## Links

A button can appear as a link with no styling by assigning `"link"` to the `type` prop.

<Playground>
  <Button palette="primary" kind="link">
    Default
  </Button>
</Playground>

## Static

To make a button non-interactable, add the `isStatic` prop.

<Playground>
  <Button isStatic>Static</Button>
</Playground>

## Disabled

To indicate a button as disabled, use the `disabled` prop.

<Playground>
  <Set>
    <Button disabled>Default</Button>
    <Button palette="primary" disabled>
      Primary
    </Button>
    <Button palette="success" disabled>
      Success
    </Button>
    <Button palette="danger" disabled>
      Danger
    </Button>
    <Button palette="warning" disabled>
      Warning
    </Button>
  </Set>
</Playground>

## Sets

To display a set of buttons, use the [`<Set>` component](/components/set).

<Playground>
  <Set>
    <Button>Previous</Button>
    <Button>Next</Button>
  </Set>
</Playground>

## Group

To group a set of buttons together, use the `<Group>` component.

<Playground>
  <Group>
    <Button>Previous</Button>
    <Button>Now</Button>
    <Button>Next</Button>
  </Group>
  <Group isVertical marginTop="xsmall">
    <Button>Previous</Button>
    <Button>Now</Button>
    <Button>Next</Button>
  </Group>
</Playground>

### Schema

```jsx
{
  base: string | Object,

  disabled: string | Object,

  // link button type styles
  link: string | Object,

  // loading button styles
  loading: string | Object,

  // static button styles

  // size styles

  }

```

# ActionButtons

## Import

`import { ActionButtons } from 'fannypack'`

## Basic Usage

The `<ActionButtons>` component renders a primary action button (e.g 'Submit'), as well as a secondary action button (e.g 'Cancel').
They can be helpful for forms, dialogs and other things that require an action.

<Playground>
  <ActionButtons onClickSubmit={() => console.log('submitted')} onClickCancel={() => console.log('cancelled')} />
</Playground>

## Colors

Action buttons can also take a color from the palette using the `palette` prop.

<Playground>
  <Set isVertical>
    <ActionButtons
      palette="text"
      onClickSubmit={() => console.log('submitted')}
      onClickCancel={() => console.log('cancelled')}
    />
    <ActionButtons
      palette="secondary"
      onClickSubmit={() => console.log('submitted')}
      onClickCancel={() => console.log('cancelled')}
    />
    <ActionButtons
      palette="danger"
      onClickSubmit={() => console.log('submitted')}
      onClickCancel={() => console.log('cancelled')}
    />
  </Set>
</Playground>

## Custom text

To modify the text of the buttons, use the `submitText` and/or `cancelText` props.

<Playground>
  <ActionButtons cancelText="Close" submitText="Save" onClickSubmit={() => console.log('submitted')} onClickCancel={() => console.log('cancelled')} />
</Playground>

## Loading

To indicate the primary button as loading, use the `isLoading` prop.

<Playground>
  <ActionButtons isLoading onClickSubmit={() => console.log('submitted')} onClickCancel={() => console.log('cancelled')} />
</Playground>

## Type

To change the type of the submit button (e.g `submit` or `reset`). Use the `type` prop.

<Playground>
  <ActionButtons type="submit" />
</Playground>

## Custom props

You can pass custom button props to action buttons using `submitProps` or `cancelProps`.

<Playground>
  <ActionButtons cancelText="Reject" submitProps={{ disabled: true }} cancelProps={{ palette: 'danger' }} />
</Playground>
