# Group

## Import

`import { Group } from 'fannypack'`

## Basic Usage

The `<Group>` component groups components together with their respective `border-radius`.

<Playground>
  <Group>
    <Box backgroundColor="primary" color="white" padding="0.5rem 1rem">
      Hello
    </Box>
    <Button>Send</Button>
  </Group>
</Playground>

## Vertical

A group can also be aligned vertically.

<Playground>
  <Group isVertical>
    <Box backgroundColor="primary" color="white" padding="0.5rem 1rem" textAlign="center">
      Hello
    </Box>
    <Button>Send</Button>
  </Group>
</Playground>

## Responsive

A group can snap from horizontal to vertical after a given breakpoint with the `verticalAt` prop.

The `verticalAt` prop accepts the following values: `"mobile"`, `"tablet"`, `"desktop"`, `"widescreen"` or `"fullHD"`.

<Playground>
  <Group verticalAt="tablet">
    <Box backgroundColor="primary" color="white" padding="0.5rem 1rem">
      Hello
    </Box>
    <Button>Send</Button>
  </Group>
</Playground>
