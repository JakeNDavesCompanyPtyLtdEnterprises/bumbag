# DialogModal

## Import

`import { DialogModal, Modal } from 'fannypack'`

## Basic Usage

The `<DialogModal>` component uses the `<Dialog>` & `<Modal>` component internally, and provides you with nice accessibility features out of the box (such as focus trapping & accessibility ids).

It is recommended to use the modal helper components (`<Modal.Show>`, `<Modal.Hide>`) when building modals to leverage accessibility features they provide like so:

<Playground>
  <Modal.Container>
    {modal => (
      <Box>
        <Button use={Modal.Show} {...modal}>
          Click me
        </Button>
        <DialogModal showCloseButton title="This is a title" {...modal}>
          Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue,
          ultrices eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum
          consectetur lectus augue sit amet justo.
        </DialogModal>
      </Box>
    )}
  </Modal.Container>
</Playground>

## Initial focus

You can leverage the initially focused element in the modal using the `initialFocusRef` render prop.

<Playground>
  <Modal.Container>
    {modal => (
      <Box>
        <Button use={Modal.Show} {...modal}>
          Click me
        </Button>
        <DialogModal showCloseButton title="This is a title" {...modal}>
          {({ initialFocusRef }) => (
            <Button use={Modal.Hide} elementRef={initialFocusRef} {...modal}>
              Close
            </Button>
          )}
        </DialogModal>
      </Box>
    )}
  </Modal.Container>
</Playground>

## Action buttons

The `<DialogModal>` component uses `<Dialog>` internally, which means you can also add in action buttons with the `showActionButtons` prop.

`DialogModal` also accepts the `actionButtonsProps` that accepts the props of the [`<ActionButtons>` component](/components/button#actionbuttons-props).

<Playground>
  <Modal.Container>
    {modal => (
      <Box>
        <Button use={Modal.Show} {...modal}>
          Click me
        </Button>
        <DialogModal
          actionButtonsProps={{ onClickSubmit: () => alert('hello!') }}
          showActionButtons
          title="This is a title"
          {...modal}
        >
          Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue,
          ultrices eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum
          consectetur lectus augue sit amet justo.
        </DialogModal>
      </Box>
    )}
  </Modal.Container>
</Playground>

## Restricting closure

By default, `DialogModal`'s can be closed by clicking 'esc' or outside the modal area. To control this behaviour, you can pass the `hideOnEsc` and/or `hideOnClickOutside` props.

<Playground>
  <Modal.Container>
    {modal => (
      <Box>
        <Button use={Modal.Show} {...modal}>
          Click me
        </Button>
        <DialogModal hideOnEsc={false} hideOnClickOutside={false} title="This is a title" showActionButtons {...modal}>
          Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue,
          ultrices eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum
          consectetur lectus augue sit amet justo.
        </DialogModal>
      </Box>
    )}
  </Modal.Container>
</Playground>

## Types

If you pass a type of `"info"`, `"success"`, `"warning"` or `"danger"` - it will add a respective icon and style the action buttons.

Using types with [alert dialogs](#alerts) can be handy!

<Playground>
  <Modal.Container>
    {modal => (
      <Box>
        <Button use={Modal.Show} {...modal}>
          Click me
        </Button>
        <DialogModal
          type="info"
          title="This is a title"
          showCloseButton
          {...modal}
        >
          Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue,
          ultrices eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum
          consectetur lectus augue sit amet justo.
        </DialogModal>
      </Box>
    )}
  </Modal.Container>
</Playground>

## Alerts

A `DialogModal` of kind 'alert' can be used to display urgent information that demands the user's immediate attention.

By default, you are not allowed to close the modal via the 'esc' or clicking outside - it requires a user response.

<Playground>
  <Modal.Container>
    {modal => (
      <Box>
        <Button use={Modal.Show} {...modal}>
          Click me
        </Button>
        <DialogModal
          kind="alert"
          type="danger"
          title="This is a title"
          {...modal}
        >
          Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse diam ipsum, cursus id placerat congue,
          ultrices eget lectus. Duis posuere, lacus sed tristique commodo, sapien turpis mollis nunc, vestibulum
          consectetur lectus augue sit amet justo.
        </DialogModal>
      </Box>
    )}
  </Modal.Container>
</Playground>
