# Rating

## Import

`import { Rating } from 'fannypack'`

## Basic Usage

The rating callback is called with `{ rating, maxRating }`.

<Playground>
  <Rating  onRate={result => console.log(result)}/>
</Playground>

## Default Rating

<Playground>

</Playground>
